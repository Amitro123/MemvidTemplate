STEP 1: Clean Up the Project
────────────────────────────────────────────────────────────────
Run this command:
    python cleanup_for_github.py

Type "yes" when prompted.

This removes:
  ✓ All generated .mp4 and .json files
  ✓ Your .env file (with API key)
  ✓ Sample documents
  ✓ Temporary files


STEP 2: Initialize Git
────────────────────────────────────────────────────────────────
Run these commands:

    git init
    git add .
    git commit -m "Initial commit: Memvid Knowledge Base Template"


STEP 3: Create GitHub Repository
────────────────────────────────────────────────────────────────
Go to: https://github.com/new

Fill in:
  • Repository name: memvid-knowledge-base
  • Description: A template for creating searchable knowledge bases from PDFs and text files using Memvid
  • Visibility: Public
  • ✅ CHECK "Template repository" (IMPORTANT!)
  • Don't add README, .gitignore, or license (we have them)

Click "Create repository"


STEP 4: Push to GitHub
────────────────────────────────────────────────────────────────
GitHub will show you commands. Use these:

    git remote add origin https://github.com/YOUR_USERNAME/memvid-knowledge-base.git
    git branch -M main
    git push -u origin main

Replace YOUR_USERNAME with your actual GitHub username!


STEP 5: Verify Template Settings
────────────────────────────────────────────────────────────────
On GitHub:
  1. Go to your repository
  2. Click "Settings" tab
  3. Scroll to "Template repository"
  4. ✅ Make sure it's checked
  5. Save if needed


STEP 6: Add Topics (Optional)
────────────────────────────────────────────────────────────────
On your repo page:
  1. Click the gear icon next to "About"
  2. Add topics: memvid, knowledge-base, pdf-search, semantic-search, ai, template, python
  3. Save


STEP 7: Test It!
────────────────────────────────────────────────────────────────
  1. Go to your repository
  2. You should see a green "Use this template" button
  3. Click it to test creating a new repo from the template

OPTIONAL: Add a License
────────────────────────────────────────────────────────────────
On GitHub:
  1. Click "Add file" → "Create new file"
  2. Name it: LICENSE
  3. Click "Choose a license template"
  4. Select "MIT License"
  5. Fill in your name
  6. Commit
